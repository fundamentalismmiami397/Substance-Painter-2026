## 📋 Common Tasks

### Starting a New Project
Select File and then New. Choose your 3D model file, which usually ends in .obj or .fbx. Set the resolution to 2048 for a balance of speed and detail. Press OK to load the model into the scene.

### Applying Materials
Open the Assets window. Drag and drop a material onto your model. The model updates immediately in the viewport. Adjust the properties of the material in the Properties window to change color, roughness, or metallic settings.

### Saving Your Work
The software saves your project as a .spp file. Use this format to keep your layers and editing history. Use the Export Textures command to create the final image files you need for other programs.

## 💡 Troubleshooting

If the program runs slow, lower the viewport resolution. You can find this setting in the top right corner of the main window. If textures look blurry, increase the resolution in the Project Settings menu. Ensure your graphics card drivers remain current to prevent rendering errors. Restart the software if the interface stops responding. If the installation fails, check that you have enough disk space and try running the installer again.
